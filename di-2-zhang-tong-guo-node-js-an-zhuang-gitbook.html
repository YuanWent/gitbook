第2章 通过node.js安装GitBook

2.1 安装GitBook

  点击开始—>运行—>输入cmd,或者Win+R输出cmd

$ npm install gitbook-cli -g

  安装成功后，执行gitbook -V查看版本信息。此命令会默认同时安装 GitBook。至此，安装成功。
